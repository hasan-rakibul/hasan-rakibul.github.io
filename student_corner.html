Open call(s) in projects

Sleep Apnea Detection System on Edge Device

Project Details: This project aims to develop a liteweight sleep apnea detection system that can be deployed on edge devices like mobile phones. Using open-access datasets, students will work on building a system that can detect apneic events during sleep using audio recordings, with a particular focus on breathing sounds. Sleep apnea is a chronic condition with severe health implications, but its prevalence is often underestimated due to the inconvenience of traditional diagnostic methods. The project's significance lies in enhancing awareness of this syndrome and making diagnosis more accessible through the use of mobile technology. The anticipated work involves data analysis, signal processing, machine learning, and optimisation for deployment on mobile devices.

Skills:

  • Data Analysis: Good understanding of data manipulation, visualisation, and interpretation.
  • Machine Learning: Proficiency with machine learning concepts, algorithms and programming (e.g., Python and relevant libraries).
  • Signal Processing: Familiarity with signal processing techniques, especially in the context of audio data.
  • Mobile App Development: Familiarity with mobile app development (e.g., iOS, Android) or willingness to learn.
